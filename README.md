# 🚀 QuickHire Backend API

This is the backend RESTful API for the **QuickHire – Simple Job Board Application**.  
Built with **Node.js, Express.js, and MongoDB**, this API handles job listings, job applications, and basic admin management functionality.

---

## 📌 API Documentation

You can test and explore all available endpoints using the official Postman documentation:

🔗 **Postman Documentation:**  
https://documenter.getpostman.com/view/46559868/2sBXcKCdXb

---

## ✨ Features

### 🌍 Public Routes
- Get all job listings (with search & filter by category/location)
- Get single job details
- Submit a job application  
  - Basic email validation  
  - Resume URL validation  

### 🔐 Admin Routes (Protected)
- Create a new job listing
- Delete an existing job listing

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv
